## 📋 項目概述

### 核心目標
為大豐集團主管群打造一個線上 Gemini AI 應用教學網站,支持：
- 四週完整課程體系（12 個課程模組）
- 83 頁投影片講義
- 互動投影片模式
- 引導式學習（開發中）

### 課程內容
- **第 1 週：** 見樹又見林（六大神器介紹）
- **第 2 週：** 超級實習生（Prompt 交辦公式）
- **第 3 週：** 深度研究與知識庫
- **第 4 週：** 沉浸式協作與多模態生成

## 🌐 技術詳情

### 項目結構
```
gemini-training-site/
├── app/
│   ├── page.tsx              # 首頁（新布局：4 列每週）
│   ├── layout.tsx            # 全局佈局
│   ├── globals.css           # 全局樣式（Brutalism）
│   └── course/
│       └── [week]/
│           └── page.tsx      # 課程週次頁面 + 投影片模式
├── lib/
│   └── course-data.ts        # 課程數據（12 模組 × 83 頁）
├── public/
│   ├── logo_TF-04.png        # 企業標誌
│   ├── logo_banner.png       # 橫幅標誌
│   └── images/               # 講義插圖（待上傳）
├── next.config.ts            # Next.js 配置
├── package.json              # 依賴配置
└── tsconfig.json             # TypeScript 配置
```

### 核心依賴
- **Next.js：** 16.2.1
- **React：** 19.2.4
- **TypeScript：** 最新

### 構建和部署
- **本地開發：** `npm run dev` → http://localhost:3000
- **生產構建：** `npm run build`
- **部署平台：** Vercel（自動 CI/CD）
- **GitHub 倉庫：** https://github.com/blacKgreYcAt/gemini-training-site

### 關鍵配置

**next.config.ts：**
```typescript
const nextConfig: NextConfig = {
  typescript: {
    ignoreBuildErrors: false,
  },
};
```

**動態路由：**
- 使用 `generateStaticParams` 預生成 /course/1, 2, 3, 4 路由
- 支持 Promise-based params（Next.js 16 特性）

**TypeScript 類型：**
- 完整的 interface 定義
- 參數類型安全：`{ params: Promise<{ week: string }> }`
